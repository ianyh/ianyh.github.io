• Two new layouts:
    • Row Layout: partitions the screen horizontally into rows of uniform height.
    • Widescreen Tall Layout: like tall layout, but uses columns in the main pane and rows in the secondary pane.
  • Performance improvements in the responsiveness of changing layouts and focusing windows.
  • Fixed an annoying flickering in windows that would frequently happen when switching spaces.
  • Added the ability to specify the arrow keys as command keys in .amethyst.
  • mouse-follows-focus now correctly works when windows are moved via Amethyst.
  • Added a new command mod2+space to reverse cycle through layouts.
  • Added a new set of commands mod2+h and mod2+l for cycling windows between screens.
  • Added better error handling for JSON errors in .amethyst.
  • Fixed a bug in which focused windows with sheets would sometimes be tiled partially offscreen.
  • Added configuration options for delaying the layout HUD.
  • Fixed visual artifacting in app icon on retina displays.
  • Fix bugs so that Amethyst now plays nicely with apps in fullscreen mode.
